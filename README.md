# CBPW
Анализ зависимостей социально-экономического развития регионов и уровня развития региональных финансовых рынков

## Pipeline:
1) [cbpw_liba] библиотека-hub с функциями в рамках текущего проекта
2) [preprocessing_stages] - стадия предобработки данных из оригинального состояния в требуемый формат
3) [vidgets] - файл с интерактивной визуализацией индексов и не только
4) [main_models] - основной блок, в рамках которого проводится анализ
